# Project Images

Place your project images in this directory with the following names:

- `vissioon.jpg` - Image for Vissioon project
- `talentsource.jpg` - Image for TalentSource project
- `doneworkspace.jpg` - Image for Done Workspace project
- `mailcub.jpg` - Image for MailCub project

**Recommended image specifications:**
- Format: JPG, PNG, or WebP
- Dimensions: 1200x800px or similar aspect ratio
- File size: Optimized (under 500KB recommended)

If images are not found, placeholder images will be automatically generated.
